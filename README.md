# Achados Promo Clube — Página GitHub Pages

Página estática simples para divulgar a vitrine de afiliados.

**Link da vitrine:**  
https://collshp.com/achadospromoclube?view=storefront

## Como publicar

1. Crie um repositório no GitHub (ex: `achadospromoclube-site`).
2. Clique em **Add file → Upload files**.
3. Envie:
   - `index.html`
   - `README.md`
4. Vá em **Settings → Pages**.
5. Em **Source**, selecione:
   - Branch: `main`
   - Pasta: `/ (root)`
6. Salve e aguarde o link do GitHub Pages.

Use esse link na bio do TikTok, Instagram e onde quiser divulgar sua vitrine.
